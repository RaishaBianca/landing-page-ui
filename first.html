Workfrom

Study Case

Workfrom is a platform connecting property owners with renters for various workplace and event needs. It offers a user-friendly solution for renters to find and secure spaces that meet their requirements. The platform's owner is committed to refreshing the design and enhancing the user experience.

Research methode : Focus Group Discussion

Duration : 3 Days

Role : UI Designer, UX Researcher

Problems & Solutions

Problems

  • The same room appears multiple times due to a logic error in designing the tagging system
  • Inconsistent content entries in the search menu.
  • Navigation that does not have a suitable hierarchy.
  • The sorting feature is not yet available, and the filter feature is incomplete.
  • There are no detailed pages for each room.
  • There is no information about the duration of the displayed prices.

Solutions

  • Each room can have more than one use case tag.
  • Maintain entry size by truncating overly long content.
  • Place the blog under the landing page.
  • Complete the filter feature and add sorting functionality.
  • Add detail pages for use case menus or rooms.
  • Add information about the duration of room or office usage.
